# AI 数据分析助手

基于 Python + Streamlit 开发的智能数据分析 Web 应用，集成 AI 大模型实现自然语言数据分析。

## 功能特点
- 上传 CSV 文件，自动生成可视化图表
- 接入 Groq API（LLaMA 3.3）进行 AI 智能分析
- 支持 Ollama + Gemma4 本地模型推理
- 一键生成数据分析报告

## 技术栈
Python · Streamlit · Pandas · Matplotlib · Groq API · Ollama · Docker · n8n
